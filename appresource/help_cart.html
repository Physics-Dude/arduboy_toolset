Arduboy Toolset - Cart Editor

This tool lets you modify FX flashcart .bin files directly. You can then reflash your Arduboy FX with the new flashcart.

So to modify the cart on your Arduboy, you might:

  • Back up the flashcart on your Arduboy first (you can use Arduboy Toolset or any other backup app)
  • Open the .bin file in the Cart Editor
  • Use the Cart Editorto add or modify the games in the cart
  • Save the .bin file
  • Upload the new .bin file to your Arduboy (you can use Arduboy Toolset or any other upload app)
