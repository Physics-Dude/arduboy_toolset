Arduboy Toolset

Arduboy toolset lets you upload + backup sketches, flashcart data, and EEPROM.

Sketches

Sketches are compiled singular programs, usually with a .hex extension. Sometimes a hex file will be embedded inside a .zip or .arduboy file. All three are accepted for upload to Arduboy. Note that Arduboy can only hold one sketch at a time, so uploading will overwrite whatever is currently there.

Flashcart

Arduboy FX and others with the "flashcart" have a special bootloader that boots before the sketch to allow you to choose one of many games on the additional flash memory (usually 16MB). The bootloader will flash the game to the "sketch" area and boot into it, just like this tool does when you upload a sketch.

You can generate flashcart .bin files by using the Cart Builder website. The bin file you download from there can be directly flashed to your Arduboy using the Flashcart upload portion of this tool

EEPROM

EEPROM is a small 1024 byte area made specifically for save game data. All games have to share the same space for saves, so they often overwrite each other. You can backup + restore this data using this tool
